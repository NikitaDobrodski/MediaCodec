# MediaCodec

Тестовое задание по алгоритмам медиакодирования для Carrot Broadcast.  
Реализация IMA ADPCM аудиокодека и MJPEG видеокодека с нуля, без использования сторонних библиотек кодирования.

---

## Требования

- Windows 10 / 11
- .NET 8.0 Runtime
- Visual Studio 2022 (для сборки)

Никаких NuGet-зависимостей на сторонние кодеки. Воспроизведение аудио использует встроенный Windows MCI (`winmm.dll`).

---

## Быстрый старт

1. Открыть `MediaCodec.sln` в Visual Studio
2. Собрать проект (`Ctrl+Shift+B`)
3. Запустить (`F5`)

## Часть 1 — Аудиокодек IMA ADPCM

### Что реализовано

- **`ImaAdpcmEncoder`** — кодирование PCM (16-bit) → IMA ADPCM
- **`ImaAdpcmDecoder`** — декодирование IMA ADPCM → PCM (16-bit)
- **`WavReader`** — чтение и запись WAV файлов (PCM, 16-bit)
- Поддержка моно и стерео

### Алгоритм

IMA ADPCM (Interactive Multimedia Association Adaptive Differential PCM) — адаптивный дифференциальный кодек. Каждый сэмпл кодируется в 4 бита (вместо 16) за счёт хранения разницы между текущим и предсказанным значением.

На каждый сэмпл:
1. Вычисляется разница `delta = sample - predictor`
2. Разница квантуется по таблице шагов (`stepTable`) → 4-битный nibble
3. Индекс шага корректируется по таблице `indexTable`
4. Предсказатель обновляется

Коэффициент сжатия: **4:1** (16-bit PCM → 4-bit ADPCM).

### Как использовать

1. Нажать **Загрузить WAV** → выбрать `.wav` файл
2. Нажать **Кодировать** → выбрать папку → сохранится `.adpcm` файл
3. Нажать **Декодировать** → выбрать папку → сохранится `.decoded.wav`
4. Сравнить исходный и декодированный WAV — качество сопоставимо с оригиналом

## Часть 2 — Видеокодек MJPEG

### Что реализовано

- **`MjpegEncoder`** — кодирование папки PNG-кадров в `.mjpeg` контейнер
- **`MjpegDecoder`** — декодирование `.mjpeg` обратно в растровые изображения
- **`JpegWriter`** — полная реализация JPEG кодирования с нуля:
  - Конвертация RGB → YCbCr
  - DCT (Discrete Cosine Transform) 8×8
  - Квантование (таблицы яркости и цветности)
  - Зигзаг-сканирование
  - RLE + кодирование Хаффмана
  - Запись JFIF-совместимого JPEG
- **`TestFrameGenerator`** — генерация тестовых PNG-кадров с анимацией

### Формат контейнера

Кастомный бинарный контейнер `.mjpeg`:

```
[4 байта]  Magic = "MJPG"
[4 байта]  Количество кадров (big-endian uint32)
--- для каждого кадра ---
[4 байта]  Длина JPEG в байтах (big-endian uint32)
[N байт]   JPEG данные
```

### Как использовать

**Вариант А — тестовые кадры:**
1. Нажать **Генерировать** → ввести количество кадров → выбрать папку
2. Будут созданы PNG-кадры с анимацией (движущийся шар, градиент, полосы)

**Вариант Б — свои кадры:**
1. Нажать **Загрузить кадры** → выбрать папку с PNG файлами
2. Кадры сортируются по имени файла

**Кодирование:**
1. Нажать **Кодировать** → выбрать папку для сохранения
2. Сохранится `output.mjpeg`

**Декодирование:**
1. Нажать **Декодировать** → выбрать папку для сохранения
2. Каждый кадр сохраняется как `frame_0001.png`, `frame_0002.png` и т.д.
3. В области предпросмотра отображается прогресс декодирования

## Часть 3 — Синхронное воспроизведение (опционально)

### Что реализовано

- **`MjpegPlayer`** — плеер с синхронизацией видео и аудио
- Синхронизация через `Stopwatch` (.NET) + Windows MCI (аудио)
- Корректная остановка при конце видео или аудио
- Поддержка воспроизведения только видео (без аудио)

### Алгоритм синхронизации

- `Stopwatch` запускается одновременно с MCI аудио
- На каждом тике таймера: `frameIdx = elapsedMs * fps / 1000`
- Если видео закончилось (`frameIdx >= frameCount`) → стоп
- Если аудио закончилось естественно → стоп
- Все MCI-команды (`stop`, `close`) выполняются на UI-потоке через `SynchronizationContext` — требование waveaudio драйвера Windows

### Как использовать

1. Загрузить WAV, закодировать аудио (ADPCM), декодировать обратно в WAV
2. Загрузить кадры, закодировать видео в MJPEG
3. Нажать **▶ Играть** — плеер запустит синхронное воспроизведение
4. **■ Стоп** — остановить в любой момент

> Для воспроизведения только видео (без аудио) — не загружать WAV, просто кодировать и играть.

## Структура проекта

```
MediaCodec/
├── Audio/
│   ├── ImaAdpcmEncoder.cs     # Кодирование PCM → ADPCM
│   ├── ImaAdpcmDecoder.cs     # Декодирование ADPCM → PCM
│   └── WavReader.cs           # Чтение/запись WAV
├── Video/
│   └── Jpeg/
│       ├── ColorSpace.cs      # RGB → YCbCr конвертация
│       ├── DctTransform.cs    # DCT 8×8
│       ├── HuffmanCoder.cs    # Таблицы и кодирование Хаффмана
│       ├── JpegWriter.cs      # JPEG кодирование (полный пайплайн)
│       ├── Quantizer.cs       # Квантование
│       ├── ZigZag.cs          # Зигзаг-сканирование
│       ├── MjpegEncoder.cs    # PNG-кадры → .mjpeg контейнер
│       ├── MjpegDecoder.cs    # .mjpeg → Bitmap кадры
│       ├── MjpegPlayer.cs     # Синхронный плеер (видео + аудио)
│       └── ControlExtensions.cs  # InvokeIfRequired хелпер
├── Form1.cs                   # Главная форма (UI логика)
├── Form1.Designer.cs          # Дизайн формы
├── Form1.resx                 # Ресурсы формы
├── Program.cs                 # Точка входа
└── TestFrameGenerator.cs      # Генерация тестовых PNG-кадров
```
